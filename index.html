Global Game Jam 2019

Scout

Scout is a 2D platform-based game where the protagonist, Scout, journeys through the land in search of their home. The player controls Scout through 3 levels, each with their own unique time and setting to help Scout. The player will encounter others, have to make decisions, and learn what it means to have a home. Scout is controlled by the left and right arrow keys along with the spacebar on the keyboard. The player's task to get Scout through the level and avoid the various obstacles. At end of each level, scout will have a chance to choose between different characters to bring along on their journey for a home.

2019 Spring UE4Jam

Crack The Code

Crack the Code is an educational game geared towards teaching people how to code based on Unreal Engine 4. You learn by playing through a set of mini-games and each one teaches users the basic fundamentals of needing to hack into someone else's computer. There are currently 3 minigames in place

  • A Typing minigame
  • A XOR minigame
  • A Rapid clicking minigame
